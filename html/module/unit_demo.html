错误提示效果

如果表单填写错误，则显示错误提示,显示2s后消失，显示时改变错误提示的内容即可，“error_tip”即为错误提示的弹出框名，复制时去unit_demo复制带缩进的js

/*点击注册按钮*/ $('.regist_button').click(function(event) { /*如果表单填写错误，执行以下js,则显示错误提示,显示2s后消失，显示时改变错误提示的内容即可*/ if(!$('.error_tip').is(":animated")){ $('.error_tip').show(100, function() { $('.error_tip').html('手机号码输入错误');//这里输入要提示的内容 $('.error_tip').animate({ opacity: 1, },1000, function() { window.setTimeout(function(){ $('.error_tip').animate({ opacity: 0, }, 1000, function() { $('.error_tip').hide(); }); }, 2000) }); }); } });
